# HTML/CSS Weiterbildung Workspace

This repository is the practice workspace for a WBS HTML/CSS course. It is meant to be browsed like a small course wiki: notes, examples, exercises, and one cleaner demo site in a single place.

The related final project is:
[eichhoernchen-wiki-projektarbeit](https://github.com/lylehmann/eichhoernchen-wiki-projektarbeit)

External course guide:
[HTML/CSS Weiterbildung course guide gist](https://gist.github.com/lylehmann/bfa4b8b342b31012bef359ecc0a95fe7)

## Start Here

If you are opening this repo on GitHub, use this order:

1. Read the course overview in [docs/README.md](docs/README.md)
2. Open the external guide gist:
   [HTML/CSS Weiterbildung course guide gist](https://gist.github.com/lylehmann/bfa4b8b342b31012bef359ecc0a95fe7)
3. Browse the example map in [docs/example-map.md](docs/example-map.md)
4. Open the main demo in [site/index.html](site/index.html)
5. Explore the smaller exercises in [exercises/](exercises)

## Repository Guide

- [docs/README.md](docs/README.md): course-style index for the notes
- [docs/example-map.md](docs/example-map.md): where to find examples for forms, tables, layout, selectors, and responsive ideas
- [docs/course-context.md](docs/course-context.md): how this repo connects to the final project
- [site/index.html](site/index.html): the clearest small end-to-end demo
- [exercises/lesson-d1/index.html](exercises/lesson-d1/index.html): basic HTML elements and form practice
- [exercises/task-d1/contact-form.html](exercises/task-d1/contact-form.html): simple contact form example
- [exercises/task-d1/sport-selection-form.html](exercises/task-d1/sport-selection-form.html): `fieldset`, `select`, and time input example
- [exercises/task-d1/table.html](exercises/task-d1/table.html): basic table example

## What This Repo Contains

- `docs/`: notes and reference material from the course
- `exercises/`: smaller hands-on files for practice
- `site/`: a cleaner mini demo site

## How to View the HTML Files

You can open the HTML files directly in your browser, or use a simple local server such as VS Code Live Server.

The best starting page is:
[site/index.html](site/index.html)

## Why This Repo Exists

This repo is the learning and preparation side of the course. The final squirrel website is the polished course deliverable, while this repository shows the building blocks behind it.
