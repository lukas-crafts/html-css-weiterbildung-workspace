# Block-Level Elements

In HTML, most elements are categorized as either **block-level** or **inline-level**. Understanding this distinction is key to creating proper layouts.

## Characteristics of Block Elements

- They always start on a new line.
- They take up the full width available (stretching out to the left and right as far as they can).
- They have top and bottom margins (unlike inline elements).

## Common Block elements

- `<div>`: A generic container for flow content.
- `<p>`: A paragraph of text.
- `<h1>` through `<h6>`: Section headings.
- `<header>`, `<footer>`, `<main>`, `<section>`, `<article>`: Semantic layout containers.
- `<ul>`, `<ol>`, `<li>`: Lists and list items.
